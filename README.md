# Data Analytics Project

## Overview

This project demonstrates an end-to-end data analytics workflow, covering data loading, cleaning, exploratory data analysis (EDA), PostgreSQL querying, dashboard creation, reporting, and presentation. The goal is to transform raw data into meaningful insights using industry-standard tools and best practices.

The project is designed to be clear, practical, and recruiter-friendly, showcasing skills relevant to data analyst and data engineering roles.

---

## Dataset

* The dataset is loaded and processed using **Python**.
* It contains structured data suitable for analysis, visualization, and SQL-based querying.
* Data cleaning steps ensure consistency, accuracy, and usability for downstream analysis.

> Note: The dataset source and schema can be found in the `data/` directory or referenced in the analysis notebooks.

* **Programming Language:** Python

* **Databases:** PostgreSQL

## Project Steps

1. **Data Loading**

   * Imported the dataset into Python using Pandas.

2. **Data Cleaning & Preprocessing**

   * Handled missing values and duplicates
   * Standardized data types and formats
   * Removed inconsistencies and outliers (where applicable)

3. **Exploratory Data Analysis (EDA)**

   * Analyzed data distributions and trends
   * Identified key patterns and correlations
   * Generated visualizations for better understanding

4. **SQL Analysis**

   * Loaded cleaned data into PostgreSQL
   * Wrote SQL queries to answer business questions
   * Performed aggregations, filtering, and joins

5. **Dashboard Development**

   * Built an interactive **Power BI dashboard**
   * Included KPIs, filters, and visual summaries

6. **Reporting & Presentation**

   * Created a detailed analytical report
   * Designed a professional PPT to summarize insights

## How to Run the Project

1. Clone the repository:

   ```bash
   git clone <repository-url>
   ```

2. Install required Python libraries:

   ```bash
   pip install -r requirements.txt
   ```

3. Run the Jupyter notebooks or Python scripts for:

   * Data cleaning
   * EDA
   * PostgreSQL data loading

4. Import the dataset into your PostgreSQL database.

5. Open the **Power BI** file to explore the dashboard.

6. Review the report and Gamma presentation for summarized insights.
